# CSF_Lab06_02240079

Description:
This folder has python codes written during Lab 6 of CSF101. The lab involves programming and analysis of various algorithms such as searches, sorts, recursion, and simple data structures.

Tasks Included:
1.Linear Search
2.Indexed (Block) Search
3.Selection Sort
4.Trace Selection Sort
5.Factorial (Iterative & Recursive)
6.Fibonacci (Iterative & Recursive)
7.Stack with Sorting
8.Queue with Searching

How to Run:
Make sure Python is installed, then run any file using:
python filename.py
Example:
python Task1_linear_search.py
